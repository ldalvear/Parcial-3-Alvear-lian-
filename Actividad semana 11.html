Herramienta de texto (T) → para escribir el texto. Herramienta de selección (F1) Cambio de fuente → menú Texto → Texto y tipografía Extensión de texto → Caja de tipo → convertir el texto en mayúsculas. Duplicar objeto (Ctrl+D / Cmd+D) → para trabajar copias. Convertir a trayecto (Shift+Ctrl+C / Trayecto → Objeto a trayecto) → convertir texto editable en trayecto editable. Menú de efectos de trayecto → Perspectiva (sección Distorsionar) → aplicar perspectiva al texto. Herramienta de edición de nodos (F2) → mover nodos blancos de los efectos de trayecto. Herramienta de caja 3D → crear una caja en perspectiva para ajustar texto a sus caras. Herramienta de conmutar ajustes (ajuste a objetos) → para alinear y encajar texto dentro de la caja. Efecto de trayecto → Porción (sección Generar) → cortar el texto en partes para ajustarlo a distintas caras. Agrupar objetos (Ctrl+G / clic derecho → Agrupar) → manejar varias letras/trayectos como un solo objeto. Herramienta de rectángulo (R) → crear rectángulos negros de fondo o con borde blanco. Cambio de color de relleno y trazo → convertir el texto en blanco, rectángulos negros, bordes blancos.

Técnica 1: Rellenar áreas delimitadas + Recorte Herramienta “Rellenar áreas delimitadas” (balde, tecla U) → para crear una selección cerrada alrededor del objeto. Trayecto → Descombinar (Shift+Ctrl+K) → separar las formas creadas. Clic derecho → Aplicar recorte → eliminar el fondo. Rectángulo de prueba → para comprobar si el fondo quedó realmente transparente Técnica 2: Vectorizar mapa de bits Duplicar imagen (Ctrl+D). Trayecto → Vectorizar mapa de bits (Shift+Alt+B) → convertir imagen raster en vectores. Usó la pestaña Colores, con opciones: número de colores, apilar, eliminar fondo. Selección de áreas azules (Ctrl + clic) → eliminar manualmente restos del fondo vectorizado. Herramienta de nodos (F2) → ajustar bordes y corregir colores mal detectados. Técnica 3: Dibujar contorno manual + Recorte Herramienta Dibujar curvas Bézier y líneas rectas (B) → trazar el contorno manualmente. Menú Relleno y borde (Shift+Ctrl+F) → asignar relleno sólido y quitar trazo. Reducir opacidad del contorno → para ver mejor la imagen debajo. Herramienta de nodos (F2) → mejorar curvas, suavizar, borrar nodos innecesarios, usar controladores Bézier. Aplicar recorte (Shift + seleccionar imagen + objeto → clic derecho → Aplicar recorte) → quitar el fondo.
